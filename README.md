# darts
非常简单易用的Restful-API测试框架

## 依赖
* nodejs
* npm
* Jasmine
* FrisbyJS

## 参数示例
```js
{
    "desc": "resthub unit-test 1", 
    "method": "GET", 
    "params": "",
    "data": {},
    "jsonBool": true,
    "url": "http://apis.qianbao.com/basicservice/v1/gateway_unit_test",
    "httpStatus": 200,
    "jsonStatus": "20000000",
  },
```
